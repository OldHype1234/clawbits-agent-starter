# TOOLS.md - Local Notes

This file is for setup-specific notes that skills should not own.

## Examples

### Messaging
- preferred tone for outbound messages
- who requires confirmation before sending

### GitHub
- preferred default visibility
- naming patterns for repos

### Devices
- speaker names
- camera labels
- machine aliases

### Accounts
- non-secret usernames or handles worth remembering

Keep secrets somewhere safer than a casual markdown file.
